# 📊 HR Analytics Dashboard | Power BI

## 📌 Project Overview

The **HR Analytics Dashboard** is an interactive Power BI project developed to analyze employee data and identify factors influencing employee attrition. This dashboard provides HR professionals and business stakeholders with valuable insights into workforce demographics, employee satisfaction, salary distribution, experience levels, and department-wise attrition trends.

The primary goal of this project is to support data-driven HR decision-making by identifying key factors affecting employee turnover and helping organizations improve employee retention strategies.

## 🎯 Project Objectives

- Analyze employee attrition trends.
- Identify departments with the highest employee attrition.
- Compare attrition across different salary slabs.
- Analyze employee demographics.
- Study attrition based on gender and experience.
- Understand the relationship between job satisfaction and employee attrition.
- Build an interactive HR dashboard for data-driven decision-making.

## 📉 Key Insights

- Administration department has the highest employee count.
- Employees earning **6–10 LPA** experience the highest attrition.
- Most employees belong to the **26–35 years** age group.
- Male employees represent approximately **60%** of the workforce.
- Job satisfaction significantly impacts employee attrition.
- Employees with lower work experience are more likely to leave the organization.
